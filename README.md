# Previsão do Tempo Global – Projeto 4

Aplicativo web desenvolvido para o **Projeto 4: Aplicativo de Previsão do Tempo**, com:

 Consumo da **OpenWeatherMap API** (chave inserida)  
 **Modo escuro/manual com botão de alternância**  
 **Design moderno, responsivo e global**  
 **Previsão atual, horária (24h) e diária (7d)**  
 **Armazenamento local de cidade e tema**

## Como usar

1. Salve os 4 arquivos em uma pasta.
2. Abra `index.html` no navegador.
3. Use o botão no canto superior direito para alternar o tema.
4. Digite qualquer cidade do mundo (ex: Luanda, Paris, Tokyo).

 **Esta versão contém chave de API.**  
 Use apenas **localmente**.  
 Não envie para GitHub público ou servidores sem proteção.

## Requisitos do Projeto 4 cumpridos

-  API REST consumida
-  JSON manipulado
-  HTML/CSS/JS puro
-  UI responsiva e atraente
-  Previsão atual + horária + diária
-  Tratamento de erros
-  Armazenamento local (`localStorage`)
-  Modo escuro preferido pelo usuário
